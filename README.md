# Al Rahman - Premium Dress Design

## Elite Tailoring Booking System

A professional booking system for Al Rahman, a premium dress design and tailoring shop located in Lad Baaz, Hyderabad Old City.

### Features

✨ **Elite Customer Booking**
- Easy appointment scheduling
- Multiple service selection (Traditional, Formal, Bridal, Casual, Custom Design, Alterations)
- Time slot availability
- Measurement tracking
- Special requests and notes

👥 **Customer Services**
- Traditional Wear (Sarees, Lehengas, Ethnic wear)
- Formal Wear (Suits, Dresses, Formal Attire)
- Bridal Collection (Bespoke bridal wear)
- Casual Wear (Everyday dresses)
- Custom Design (Personalized creations)
- Alterations (Professional modifications)

🎨 **Design Features**
- Luxury premium aesthetic
- Responsive mobile-friendly design
- Gold accent branding (#d4af37)
- Smooth animations and transitions
- Professional gallery showcase
- Contact information section

💻 **Technology Stack**
- HTML5
- CSS3 (with responsive grid layout)

- Ready for backend integration

### Getting Started

1. Clone the repository
2. Open `index.html` in a web browser
3. Start booking appointments!

### Booking Flow

1. Customer enters personal information (Name, Phone, Email)
2. Selects preferred service type
3. Chooses appointment date and time
4. Provides body measurements (optional)
5. Adds special requests or design ideas
6. Submits booking
7. Receives confirmation message
